Privacy Policy – Virurama

Last updated: May 2026

Virurama does not collect any sensitive personal data. The game uses an external service (Supabase) to store basic gameplay information required for leaderboards and in-game statistics.

Data Collected

The game stores the following non-sensitive data:

  • Automatically generated UserID (Unity Authentication)
  • Player name (nickname)
  • Score and leaderboard data
  • Number of completed cycles
  • Basic gameplay statistics

These data are used solely to:

  • Display leaderboards
  • Save gameplay progress
  • Improve the game experience

Data Storage

All data are stored securely on Supabase, a modern and compliant cloud database service.

Data Sharing

The collected data are not shared with third parties and are not used for:

  • Marketing
  • Profiling
  • Advertising
  • Commercial analytics

User Rights

Players may request:

  • Deletion of their data
  • Modification of their nickname
  • Information about stored data

Contact for requests: fernandomenta@gmail.com

Steam

All account, purchase, cloud save, and Steam statistics features are handled directly by Steam.
